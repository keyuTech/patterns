# Patterns/设计模式
本项目是对于 [Patterns/设计模式](https://www.patterns.dev/) 的一系列笔记 or 部分翻译，一些文件中会包含额外的示例，同样来自于 [Patterns/设计模式](https://www.patterns.dev/)

设计模式是软件开发的基本组成部分，因为它们为软件设计中经常出现的问题提供了典型的解决方案。 设计模式不是提供特定的代码，而是仅仅是可用于以优化方式处理重复出现的主题的概念。
